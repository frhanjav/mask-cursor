To install the required packages:

```bash
npm i
```

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

uses: sass and motion (prev framer-motion)

```bash
npm i sass
```

```bash
npm i motion
```
